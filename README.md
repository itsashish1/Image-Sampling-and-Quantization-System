# 📄 AI-Powered Document Scanner and Image Processing System

## 📌 Overview
This project implements a complete document scanner system using Python and OpenCV.  
It performs image acquisition, resolution analysis, gray-level quantization, and quality evaluation for document processing and OCR readiness.

---

## 🚀 Features

✔ Image Acquisition from document images  
✔ Grayscale conversion and resizing  
✔ Image Sampling (Resolution Analysis)  
✔ Image Quantization (Gray Level Reduction)  
✔ Quality Observation and OCR suitability analysis  
✔ Automatic output saving system  

---

## 🧠 Tasks Implemented

### 🔹 Task 1: System Introduction
Displays welcome message and system description.

### 🔹 Task 2: Image Acquisition
- Load document image
- Resize to 512×512
- Convert to grayscale
- Display original vs grayscale

### 🔹 Task 3: Image Sampling
- High resolution (512×512)
- Medium resolution (256×256)
- Low resolution (128×128)
- Upscaling for visualization
- Comparison of clarity and edges

### 🔹 Task 4: Image Quantization
- 256 gray levels (8-bit)
- 16 gray levels (4-bit)
- 4 gray levels (2-bit)
- Visualization of quantization artifacts

### 🔹 Task 5: Quality Analysis
- Loss of fine text details
- Readability degradation
- OCR suitability evaluation
- Single comparison figure output
